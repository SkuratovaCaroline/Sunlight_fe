# Sunlight Cinema - Frontend

## Описание проекта
Fronted-часть веб-сервиса для бронирования билетов в сеть кинотеатров Sunlight cinema.

##  Стек используемых технологий
- **Frontend**: JavaScript (ES6+), HTML5, CSS3
